# Install the Hypervisor

First, you'll need to download the DreamNetwork Hypervisor's Launcher:<br/>
```wget https://cdn.dreamnetwork.cloud/hypervisor/launcher/1.0.0/dn_launcher.jar```

Give it some permissions<br/>
```chmod +x dn_launcher.jar```

Then launch it:<br/>
```java -jar dn_launcher.jar```
